# Incident Logger

## Description

A beginner Python cybersecurity script that collects and classifies security incidents based on severity.

## Features

- Takes number of incidents as input
- Collects analyst details
- Collects target information
- Classifies incidents as:
  - High
  - Medium
  - Low

## Concepts Used

- Variables
- User Input
- Loops
- Conditional Statements
- String Methods

## Technology

- Python 3.10

## Future Improvements

- Save incidents into a file
- Add timestamps
- Create incident reports

# Incident Logger v2 

## Description

A beginner-friendly Python cybersecurity script that collects incident details and classifies the incident severity.

This is the second version of the Incident Logger project, improved from a basic script into a modular program using Python functions.

## Features

- Collects analyst information
- Collects target information
- Accepts incident severity level
- Classifies incidents based on severity
- Generates incident status
- Uses reusable functions for better code organization

## Severity Classification

| Severity | Action |
|----------|--------|
| High | Immediate Investigation |
| Medium | Review |
| Low | Monitor |

## Concepts Used

- Python Functions
- Function Parameters
- Return Values
- User Input
- Conditional Statements (`if/elif/else`)
- String Methods
- Tuples


## Technologies

- Python 3.10

## Learning Outcome

Through this project, I learned how to structure a Python program using functions and how different parts of a security workflow can be separated into reusable components.
