# Incident Logger

## Description

A beginner Python cybersecurity script that collects and classifies security incidents based on severity.

## Features

- Takes number of incidents as input
- Collects analyst details
- Collects target information
- Classifies incidents as:
  - High
  - Medium
  - Low

## Concepts Used

- Variables
- User Input
- Loops
- Conditional Statements
- String Methods

## Technology

- Python 3.10

## Future Improvements

- Save incidents into a file
- Add timestamps
- Create incident reports
