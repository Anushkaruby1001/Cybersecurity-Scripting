# Security Tools Inventory 

A simple Python program to manage a list of cybersecurity tools.

## Features

- Display security tools
- Add new tools
- Search for tools
- Count total tools

## Tools Used

- Python 3
- Lists
- Functions
- User Input

## How to Run

1. Clone the repository
2. Run the Python file:

```
python security_tools_inventory.py
```

## Example Output

```
Security Tools:
- Wireshark
- Burp Suite
- Nmap

Nmap is available

Total Security Tools: 5
```
